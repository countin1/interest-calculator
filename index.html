利息计算器使用说明

单利计算：利息 = 本金 x 年利率 x 年数。适用于定期存款、国债等固定收益产品。

复利计算：支持按年、半年、季度、月复利。适用于理财产品、基金定投等复利场景。

等额本息：每月还款金额固定，适用于房贷、车贷、消费贷等分期还款场景。可查看完整还款计划表。

常见利率参考

2024年房贷利率：约 3.0%-3.5%（LPR）

银行定期存款：1年约 1.5%，3年约 2.0%，5年约 2.0%

公积金贷款：约 2.85%

公式说明

单利公式：A = P x (1 + r x t)

复利公式：A = P x (1 + r/n)^(n x t)

等额本息月供：M = P x r x (1+r)^n / ((1+r)^n - 1)

其中 P=本金，r=年利率，t=年数，n=复利频率，M=月供
